# Python-Fruit-Shop-Management-System
A console-based Python project that simulates a fruit shop inventory and billing system.

## Features

- Shopkeeper Login
- Add Items to Inventory
- Remove Items from Inventory
- View Inventory
- Customer Cart Management
- Billing System
- Profit Calculation
- Input Validation

## Technologies Used

- Python

## How to Run

```bash
python fruit_shop_management_system.py
```
